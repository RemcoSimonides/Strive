Why is the app not in the play or app store?

Main reasons being that it costs time and thus money. It is more difficult to release new versions if the app is in those stores too. There would be more code to maintain and new releases would have to be approved by Apple. Also, the users have to update their app regularly. With an installable website like this, it updates automatically for the user while also feeling like a real app.
